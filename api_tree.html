Arbol de rutas de esta API dedicada a guardar
datos generados por arduino o raspberry
-----------------------------------------------------
  - v1/data/{id}
 
      - GET: Last 60 logs
      - GET + {id}
      - POST 
      - PATCH + {id}
      - DELETE + {id}
  - v1/data/get-all
 
      - GET: All data POSTed
  - v1/data/getLast
 
      - GET: Last data POSTed
  - v1/data/lastUpdated
 
      - GET: Time of last update
