api2.orozcoh.com:4000

Arbol de rutas de esta API dedicada a guardar
datos generados por arduino o raspberry
-------------------------------------------------------------------------------
PUBLIC:
 
  - api2.orozcoh.com:4000/v1/data 
      - GET: Last 60 logs

  - api2.orozcoh.com:4000/v1/data/{timestamp} 
      - GET: Get the data of the timestamp given

  - api2.orozcoh.com:4000/v1/data/getAll 
      - GET: All data POSTed

  - api2.orozcoh.com:4000/v1/data/getLast 
      - GET: Last data POSTed

  - api2.orozcoh.com:4000/v1/data/lastUpdated 
      - GET: Time of last update
--------------------------------------------------------------------------------

ALLOWED USERS:
 
  - api2.orozcoh.com:4000/v1/data
 
      - POST: Only for allowed devices with API_KEY 
  - api2.orozcoh.com:4000/v1/data/{timestamp}
 
      - PATCH: Update info (in this data logger, this should not be possible)
      - DELETE: Delete selected data (in this data logger, this should not be possible)
